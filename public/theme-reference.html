Implementation Notes

  • Themes applied via data-theme attribute on root element
  • CSS variables defined in src/index.css under @layer base
  • Theme selector component: src/components/plan/ThemeSelector.tsx
  • All color values use HSL format without hsl() wrapper in CSS variables
  • Tailwind maps variables via tailwind.config.ts using hsl(var(--token)) pattern
  • Māori Kaupapa and Pasifika themes use body::before pseudo-element for background patterns
  • Google Fonts (Bangers, Pacifico) must be loaded externally for Crazy Colours and Random Mix-up themes
